# Основы параллельных вычислений — Лабораторная работа 1

## Постановка задачи
Написать программу на языке C/C++ для перемножения двух квадратных матриц.

- Входные данные: файл(ы) со значениями исходных матриц.
- Выходные данные: файл со значениями результирующей матрицы, время выполнения, объем задачи.
- Обязательное требование: автоматизированная верификация результатов с помощью сторонних библиотек/ПО (в проекте используется Python + NumPy).

## Текущая структура проекта
- `matrixmult/matrixmult.cpp` — C++ программа перемножения матриц.
- `matrixmult/matrixmult.vcxproj` — проект Visual Studio.
- `matrix_generate.py` — генерация входных матриц `A_n` и `B_n`.
- `verify_matrix_mult.py` — автоматическая верификация `C_n == A_n @ B_n`.
- `plot_matrixmult_time.py` — построение графика зависимости времени от размера матрицы.
- `data/` — входные/выходные матрицы и график.

## Требования
- Windows + Visual Studio (MSVC).
- Python 3.
- Python-библиотеки:
  - `numpy`
  - `matplotlib`

Установка библиотек:
```bash
py -3 -m pip install numpy matplotlib
```

## Шаг 1. Сгенерировать матрицы
```bash
py -3 matrix_generate.py
```

После запуска в `data/` появятся файлы:
- `A_10.txt`, `B_10.txt`, ...
- `A_1000.txt`, `B_1000.txt`.

## Шаг 2. Запустить C++ программу
1. Открыть `matrixmult/matrixmult.vcxproj` в Visual Studio.
2. Собрать проект (например, `Debug | x64`).
3. Запустить `matrixmult.exe`.

Программа автоматически:
- находит размеры `n` по файлам `A_*.txt` в `data/`,
- читает пары `A_n.txt` и `B_n.txt`,
- считает `C_n = A_n * B_n`,
- сохраняет `C_n.txt` в `data/`,
- выводит время и объем задачи (`2 * n^3` операций).

Пример строки вывода:
```text
N=100 | Time: 14 ms | Task volume: 2000000 operations | Saved: data/C_100.txt
```

## Шаг 3. Верификация результатов (NumPy)
```bash
py -3 verify_matrix_mult.py
```

Ожидаемый вывод:
```text
n=10: True
n=100: True
...
```

## Шаг 4. Построить график времени
```bash
py -3 plot_matrixmult_time.py
```

Результат:
- `data/time_vs_size.png` — график зависимости времени перемножения от размера матрицы.

## Примечания
- Формат файлов матриц: текст, целые числа, разделенные пробелами.
- Если для какого-то размера верификация дает `False`, нужно пересчитать `C_n` запуском `matrixmult.exe`.
